This repository contains Bib(La)TeX libraries to demonstrate JabRef's features.

## ðŸ« Chocolate Library: How to add PDFs

To populate the Chocolate demonstration library with research artifacts, follow these steps:

### Method 1: Drag and Drop
1. Locate your PDF file on your computer.
2. Drag and drop it directly onto the relevant entry in the JabRef table.

![Drag and Drop Method](./assets/drag_and_drop.jpg)

### Method 2: Manual Attachment
1. Right-click the desired entry and select **Attach file**.
2. Select your PDF from the file browser and click **Open**.

![Manual Attachment](./assets/manual.png)
![File Selection](./assets/manual-2.png)
